# ⏸️ Windows Update 延期工具

這是一個基於 **Python** 開發的 Windows 更新延期工具，它提供一個簡單的現代化UI，讓使用者可以快速調整指定延期日期。

## 🖥️ 執行畫面

> ![Windows Update 延期工具](https://raw.githubusercontent.com/zz22558822/IMG/refs/heads/main/Windows_Update_Delay/Windows_Update_Delay.png)  
> **特色：** 日期選擇器、快速選擇、狀態提示。

---

## 📦 功能特色

* **📅 自訂日期：** 提供日期選擇器，精確選擇暫停更新的結束日期。
* **⏱️ 快速選擇：** 內建「3 個月」、「6 個月」、「1 年」及「100 年」等快捷按鈕。
* **🚫 取消暫停：** 提供「取消暫停」功能，一鍵清除延期設定，恢復自動更新。

---

## 🚀 環境需求

* **💻 作業系統：** Windows 10、Windows 11
* **⚠️ 權限要求：** 正常狀況下無須管理員權限即可，若顯示需管理員時請依指示提升為**管理員權限**。
